# 🌐 Browser Battle – University Website Challenge

A fully responsive, production-ready **University Website Portal** built as part of the **Browser Battle Hackathon Challenge**. This project replicates the structure, usability, and experience of a real-world university website with modern UI/UX and scalable architecture.

---

## 🚀 Project Overview

**Browser Battle** challenges developers to design and build a complete university website that:

* Mimics real-world university portals
* Provides structured academic and institutional information
* Delivers a seamless and responsive user experience
* Is deployment-ready with clean architecture

This project goes beyond static websites by integrating **AI-powered navigation and command-based interaction**.

### ⚡ Smart Navigation System

#### ⌘ / Ctrl + K Command Menu

* Press **⌘ + K (Mac)** or **Ctrl + K (Windows/Linux)**
* Instantly opens a **Spotlight-style Command Menu**
* Quickly navigate across pages without clicking through UI
* Provides a **desktop-like experience inside the browser**

---

### 🤖 AI-Powered Navigation (Gemini Integration)

This project integrates **Google Gemini API** to enable intelligent interaction:

#### 🧠 How it works:

1. **User Input:** User types a natural query (e.g., *"Show placement stats"*)
2. **Intent Detection:** Gemini processes the query and understands intent
3. **Smart Response:**

   * Navigates to the correct page
   * Suggests relevant sections
4. **Fallback System:** Ensures functionality even if API is unavailable

#### 💡 Example Queries:

* "Take me to admissions"
* "Show departments"
* "Where are placement stats?"

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/coder-joshi/BrowserBattle.git
cd College_Website
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Add Gemini API Key

Create a `.env` file:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

---

### 4️⃣ Run the Project

```bash
npm run dev
```
